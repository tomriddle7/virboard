# virboard

![virboard](https://github.com/tomriddle7/virboard/blob/main/public/OpenGraph.png)

## V-Tuber Spot Board

## Key Features
 - Interactive Maps: Smooth map manipulation and marker navigation based on React-Leaflet.

 - Detail Modal: Stylish UI with full-screen hero images and details.

 - Direct Landing: Immediate sharing and automatic modal of specific events via URL query string.

 - Mobile Optimization: Provides an efficient information search environment with BottomDrawer.

 - Smart Alignment: Elaborate data priority alignment based on progress and image presence or absence.

 - SEO & Sharing Optimization: Complete social media sharing preview support via Twitter Card and OpenGraph.

## Tech Stack
 - Framework: React (Vite)

 - Language: TypeScript

 - Styling: Tailwind CSS

 - State Management: Jotai

 - Routing: React Router Dom

 - Map Engine: React-Leaflet

 - Hosting: GitHub Pages
